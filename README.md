# Sign In - Test A11y

A clean, accessible Sign In form matching the Figma design.

## Features

- Semantic HTML with proper form structure
- Accessible: labels, autocomplete, focus states, ARIA where needed
- Responsive layout
- Hover and focus styles for better UX

## Run

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

Then visit http://localhost:3000
